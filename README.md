## Getting Started

### Prerequisites

* npm
  `npm install npm@latest -g`

### Installation

1. Clone the repo
    `git clone https://github.com/[your_username]/[project-name.git]`
2. Install NPM packages
   `npm install`
3. Enter your TOKEN in config.js
